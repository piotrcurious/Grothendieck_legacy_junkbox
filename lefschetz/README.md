# Lefschetz Hyperplane Theorem

Documentation and explorations of the Lefschetz hyperplane theorem in complex projective varieties.

## Overview
The theorem states that for a complex $n$-dimensional projective variety $X$, slicing by a generic hyperplane $H$ preserves the topology in degrees below $n-1$. This directory provides intuitive explanations and "unpacked" versions of the theorem to make it accessible to non-topologists.

## Files
- `lefschetz_hyperplane.md`: A bottom-up explanation of the machinery involved (homology, cohomology, homotopy) and the geometric meaning of the theorem.
- `hyperplane.md`: Additional notes on the theorem's application.
