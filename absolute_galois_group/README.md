# Absolute Galois Group Explorations

This directory explores advanced concepts from field theory and algebraic geometry, specifically the Absolute Galois group $Gal(\overline{\mathbb{Q}}/\mathbb{Q})$, and their theoretical applications in data compression and representation.

## Contents
- `absolute_galois_group_1.md`: Theoretical background on Absolute Galois groups and their potential link to compression via cohomology operations.
- `compressor/`: A deep hierarchy of experimental compression algorithms leveraging algebraic structures, quadtree decompositions, and canonical shape spaces.

## State
Experimental/Theoretical. The project contains highly abstract C++ and Python implementations attempting to map data into algebraic manifolds for more efficient encoding.
